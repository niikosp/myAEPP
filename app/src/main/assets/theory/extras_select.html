Έχουμε 4 (ενδεικτικές) χρήσεις της εντολής ΕΠΙΛΕΞΕ

1. Πρόγραμμα που δέχεται έναν αριθμό και εμφανίζει αν είναι άρτιος ή περιττός.

ΔΙΑΒΑΣΕ Χ

ΕΠΙΛΕΞΕ Χmod2

ΠΕΡΙΠΤΩΣΗ 0

ΓΡΑΨΕ "Άρτιος"

ΠΕΡΙΠΤΩΣΗ ΑΛΛΙΩΣ

ΓΡΑΨΕ "Περιττός"

ΤΕΛΟΣ_ΕΠΙΛΟΓΩΝ

2. Πρόγραμμα που δέχεται έναν αριθμό και εμφανίζει αν είναι μονοψήφιος(>0) άρτιος ή περιττός.

ΔΙΑΒΑΣΕ Χ

ΕΠΙΛΕΞΕ Χ

ΠΕΡΙΠΤΩΣΗ 2,4,6,8

ΓΡΑΨΕ "Μονοψήφιος θετικός άρτιος"

ΠΕΡΙΠΤΩΣΗ 1,3,5,7,9

ΓΡΑΨΕ "Μονοψήφιος θετικός περιττός"

ΠΕΡΙΠΤΩΣΗ ΑΛΛΙΩΣ

ΓΡΑΨΕ "Σφάλμα"

ΤΕΛΟΣ_ΕΠΙΛΟΓΩΝ

3. Πρόγραμμα που δέχεται τον βαθμό ενός μαθητή και εμφανίζει:

  1. "Ανεπαρκώς" αν 0 <= ΒΑΘΜΟΣ <=10
  2. "Πολύ καλά" αν 10 < ΒΑΘΜΟΣ <=18
  3. "Άριστα" αν 18< ΒΑΘΜΟΣ <= 20

! Αρχικά κάνουμε έλεγχο εγκυρότητας για το δεδομένο εισόδου

ΑΡΧΗ_ΕΠΑΝΑΛΗΨΗΣ

ΔΙΑΒΑΣΕ ΒΑΘΜΟΣ

ΜΕΧΡΙΣ_ΟΤΟΥ ΒΑΘΜΟΣ>=0 ΚΑΙ ΒΑΘΜΟΣ<=20

ΕΠΙΛΕΞΕ ΒΑΘΜΟΣ

ΠΕΡΙΠΤΩΣΗ <=10

ΓΡΑΨΕ "Ανεπαρκώς"

ΠΕΡΙΠΤΩΣΗ <=18

ΓΡΑΨΕ "Πολύ καλά"

ΠΕΡΙΠΤΩΣΗ ΑΛΛΙΩΣ

ΓΡΑΨΕ "Άριστα"

ΤΕΛΟΣ_ΕΠΙΛΟΓΩΝ

4. Πρόγραμμα που δέχεται τη βαθμολογία ενός φοιτητή(στα 100) και εμφανίζει:

  1. "Ανεπαρκώς" αν 0 <= ΒΑΘΜΟΣ <=49
  2. "Πολύ καλά" αν 50 <= ΒΑΘΜΟΣ <=90
  3. "Άριστα" αν 90< ΒΑΘΜΟΣ <= 100
  4. Αλλιώς εμφανίζει "Σφάλμα"

ΓΡΑΨΕ "Πολύ καλά"

ΠΕΡΙΠΤΩΣΗ 91..100

ΓΡΑΨΕ "Άριστα"
