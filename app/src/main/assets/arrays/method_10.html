Ταξινόμηση ευθείας εισαγωγής - Insertion sort

Η ταξινόμηση ευθείας εισαγωγής (insertion sort) είναι ένας απλός αλγόριθμος που ταξινομεί δεδομένα παίρνοντας ένα-ένα τα στοιχεία και τοποθετώντας τα στην κατάλληλη θέση μέσα σε μια ήδη ταξινομημένη υπολίστα, ξεκινώντας από το δεύτερο στοιχείο.

Εφαρμογή

Ταξινομούμε σε αύξουσα σειρά τα στοιχεία ενός πίνακα A με n στοιχεία.

ΓΙΑ i ΑΠΟ 2 ΜΕΧΡΙ n

temp <- A[i]

j <- i-1

ΟΣΟ temp < A[j] ΚΑΙ j>0 ΕΠΑΝΑΛΑΒΕ

A[j+1] <- A[j]

j <- j-1

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

A[j+1] <- temp

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
