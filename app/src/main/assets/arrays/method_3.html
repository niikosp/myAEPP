Σειριακή αναζήτηση

Η σειριακή αναζήτηση είναι η πιο απλή μέθοδος αναζήτησης.

Αρχικά θα δούμε τη σειριακή αναζήτηση σε μονοδιάστατο πίνακα:

! Αναζήτηση στοιχείου key σε μονοδιάστατο πίνακα ΠΙΝΑΚΑΣ[30]

! 1η περίπτωση: Μας ενδιαφέρει να βρούμε το στοιχείο μία φορά

i <- 1

ΒΡΕΘΗΚΕ <- ΨΕΥΔΗΣ

ΘΕΣΗ <- 0

ΟΣΟ (i<=30) ΚΑΙ (ΒΡΕΘΗΚΕ =ΨΕΥΔΗΣ) ΕΠΑΝΑΛΑΒΕ

ΑΝ ΠΙΝΑΚΑΣ[i] = key ΤΟΤΕ

ΘΕΣΗ <- i

ΓΡΑΨΕ "ΒΡΕΘΗΚΕ ΣΤΗ ΘΕΣΗ: ", i

ΒΡΕΘΗΚΕ <- ΑΛΗΘΗΣ

ΑΛΛΙΩΣ

i <- i+1

ΤΕΛΟΣ_ΑΝ

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

! 2η περίπτωση: Μας ενδιαφέρει να βρούμε πόσες φορές υπάρχει το στοιχείο

ΠΛΗΘΟΣ <- 0

ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 30

ΑΝ ΠΙΝΑΚΑΣ[i] = key ΤΟΤΕ

ΠΛΗΘΟΣ <- ΠΛΗΘΟΣ+1

ΓΡΑΨΕ "ΒΡΕΘΗΚΕ ΣΤΗ ΘΕΣΗ: ", i

ΤΕΛΟΣ_ΑΝ

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

ΓΡΑΨΕ "ΤΟ ΣΤΟΙΧΕΙΟ key ΒΡΕΘΗΚΕ ΣΕ ", ΠΛΗΘΟΣ, "ΘΕΣΕΙΣ"

Ας δούμε τώρα τη σειριακή αναζήτηση σε δισδιάστατο πίνακα

! Αναζήτηση στοιχείου key σε δισδιάστατο πίνακα ΠΙΝΑΚΑΣ[30,50]

! 1η περίπτωση: Μας ενδιαφέρει να βρούμε το στοιχείο μία φορά

ΒΡΕΘΗΚΕ <- ΨΕΥΔΗΣ

i <- 1

ΟΣΟ i<=30 ΚΑΙ ΒΡΕΘΗΚΕ=ΨΕΥΔΗΣ ΕΠΑΝΑΛΑΒΕ

J <- 1

ΟΣΟ j<=50 ΚΑΙ ΒΡΕΘΗΚΕ=ΨΕΥΔΗΣ ΕΠΑΝΑΛΑΒΕ

ΑΝ ΠΙΝΑΚΑΣ[i,j] = key ΤΟΤΕ

ΓΡΑΨΕ "ΒΡΕΘΗΚΕ ΣΤΗ ΘΕΣΗ: ", i,j

ΒΡΕΘΗΚΕ <- ΑΛΗΘΗΣ

ΤΕΛΟΣ_ΑΝ

j <- j+1

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

i <- i+1

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

! 2η περίπτωση: Μας ενδιαφέρει να βρούμε πόσες φορές υπάρχει το στοιχείο

ΠΛΗΘΟΣ <- 0

ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ 30

ΓΙΑ j ΑΠΟ 1 ΜΕΧΡΙ 50

ΑΝ ΠΙΝΑΚΑΣ[i,j]=key ΤΟΤΕ

ΓΡΑΨΕ "ΒΡΕΘΗΚΕ ΣΤΗ ΘΕΣΗ: ", i,j

ΠΛΗΘΟΣ <- ΠΛΗΘΟΣ +1

ΤΕΛΟΣ_ΑΝ

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ

ΑΝ ΠΛΗΘΟΣ=0 ΤΟΤΕ ΓΡΑΨΕ "ΔΕΝ ΒΡΕΘΗΚΕ ΠΟΥΘΕΝΑ ΤΟ ΣΤΟΙΧΕΙΟ"
